## Features

- **Curtain opening animation** — two fabric curtains split apart to reveal the invitation
- **Floating rose petals** — ambient petal animation after the curtains open
- **Live countdown timer** — days, hours, minutes, seconds until the wedding
- **Scroll-reveal animations** — sections fade in as you scroll
- **Interactive map** — square OpenStreetMap window showing the venue location (no API key required)
- **Fully responsive** — optimised for mobile, tablet, and desktop
- **GitHub Pages ready** — zero build step, pure static files

---

## Deployment (GitHub Pages)

1. Push all files to the `main` branch.
2. Go to **Settings → Pages**.
3. Set **Source** to `main` branch, `/ (root)` folder.
4. Save — the site will be live at the URL shown within a minute.
